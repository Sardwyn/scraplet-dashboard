## What is Scraplet Broadcast Studio?

Scraplet Broadcast Studio (SBS) is a live streaming and production toolkit designed for professional and semi-professional streamers. It touches every point of a live broadcast — from overlays and alerts to chat management, production control, and AI assistance.

SBS is built for streamers who treat their stream like a show. It's not a casual tool — it's a production system.

---

## Core Philosophy

**The show must go on.** Every feature in SBS exists to make live productions smoother, more interactive, and more professional. Scrapbot, the AI layer, embodies this — it's not a chatbot, it's a production intelligence system.

---

## Platform Components

### Scraplet Dashboard
The central hub. Accessible at your Scraplet URL. Contains:
- Stream analytics and metrics
- Integration management (Kick, Discord, etc.)
- Overlay editor
- Alert configuration
- Scrapbot settings
- Community management tools

### Overlay Editor
A canvas-based editor for building on-screen graphics. Features:
- Drag-and-drop element placement
- Live data bindings (connect elements to real-time data)
- Animation system with enter/exit presets
- Component library for reusable elements
- Widget system for live data sources (Stake Monitor, etc.)
- Browser source output for OBS

### Showrunner Controller
A production control surface accessible at `/studio`. Features:
- Rundown editor with timed segments
- OBS scene switching
- Overlay operator panel
- Teleprompter
- Camera/audio/lighting controls
- Scrapbot status panel
- Live stats display

### Scrapbot
The AI production intelligence layer. Available in:
- Discord (primary control layer)
- Kick chat (community-facing)

Capabilities:
- Live stats and engagement tracking
- Slot suggestions and gambling awareness
- Image generation (Discord)
- Stream awareness and pivot suggestions
- Chat integration and light moderation
- Production intelligence and analysis

### Kick Integration
- Real-time chat ingestion
- Webhook event processing (follows, subscriptions, raids)
- Token management with automatic refresh
- Chat bot functionality via Scrapbot

### Discord Integration
- Scrapbot Discord bot
- Community sync
- Stream debrief delivery
- Production team control layer

## Hierarchy

SBS serves a production hierarchy:
1. **Streamer** — primary authority, all decisions ultimately theirs
2. **Moderators** — trusted team members with elevated access
3. **Community/Chat** — viewers with standard access

Scrapbot respects this hierarchy. It serves the production, not individual viewers.

---

## Key Differentiators

**Production-first design** — SBS is built around the concept of a live show, not just a stream. The showrunner, rundown system, and production controls reflect this.

**AI integration** — Scrapbot is not bolted on. It's woven into the production workflow — tracking metrics, surfacing insights, reacting to moments.

**Gambling-aware** — SBS is built with casino/gambling content in mind. Slot awareness, win/loss tracking, and gambling-specific features are first-class.

**Extensible** — The widget system allows new data sources to be added to the overlay editor. The knowledge base and RAG system allow Scrapbot's expertise to grow without retraining.

---

## Getting Started

1. Connect your Kick account via Dashboard → Integrations → Kick
2. Connect your Discord bot via Dashboard → Integrations → Discord
3. Build your first overlay in the Overlay Editor
4. Add the overlay as a Browser Source in OBS
5. Set up your showrunner rundown for your first show
6. Invite Scrapbot to your Discord server
